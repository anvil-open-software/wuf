Quick Start Application

The Quick Start application, which is a part of the WUF repos at https://github.com/anvil-open-software, is intended to be used as a baseline, plain-vanilla application for quickly creating a new web-based application from scratch based on WUF.

The Quick Start application makes no assumptions about any other technology, including the backend. The application allows the developers to start with a bare-bones version of the application and then add any backend, datastore, authentication, or services needed.

To create a new application based on the Quick Start application:

  1. Follow the README.md instructions at https://github.com/anvil-open-software/wuf-quick-start/blob/master/README.md.
  2. Once you have a new application installed locally, create a gitlab repo to host the source code for your new application.
  3. You can now begin adding and using the components from the "Components" section of this Living Style Guide to build your applicationAngular Application Core Framework Module.

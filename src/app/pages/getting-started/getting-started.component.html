Quick Start Application (Minimal)

There are two versions of the Quick Start application:

  • Quick Start (minimal) - This one
  • Quick Start (complete)

The Quick Start (Minimal) application, which is a part of the WUF repos at https://gitlab.dematic.com/cloud_visualization_services, is intended to be used as a baseline, plain-vanilla application for quickly creating a new web-based application from scratch.

The minimal version of the Quick Start application is the most basic and makes no assumptions about any other technology, including the backend. This version allows the developers to start with a bare-bones version of the application that is compatible with any backend, datastore, or services.

To create a new application based on the Quick Start application:

  1. Follow the README.md instructions at https://gitlab.dematic.com/cloud_visualization_services/angular-quickstart-app-minimal/blob/master/README.md.
  2. Once you have a new application installed locally, create a gitlab repo to host the source code for your new application.
  3. You can now begin adding and using the components from the "Components" section of this Living Style Guide to build your applicationAngular Application Core Framework Module.

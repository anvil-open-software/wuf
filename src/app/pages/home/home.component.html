Welcome to the WUF Living Style Guide

The Web UI Framework (WUF) is an open source framework based on Angular designed to help developers quickly and easily create applications built on a modern platform, web standards, and best practices.

WUF is based on the following technologies:

  • Node.js
  • Angular 2+
  • Typescript
  • Material Design
  • SCSS
  • HTML/CSS for Front-end layouts that are fully responsive
  • It also includes support for:
  • Web components
  • Polymer components

WUF was initially developed internally by Dematic, Inc. for use in its own applications, but it has since been released to the open source community under the MIT license. Packages are published under the @anviltech scope into the public NPM registry. Please feel free to use WUF as you see fit. We also welcome code contributions!

The WUF is comprised of 2 repos:

  1. The WUF repo. This repo holds the Living Style Guide (this application) as well as the Library of Components, each of which is published independently to the public NPM registry.
  2. The WUF Quick Start Application repo.

This application, the Living Style Guide, will serve as a living, self-documenting application that will introduce developers to WUF, demonstrate the use of various components available within the Library of Components, and explain how to build applications based on WUF (i.e., the Quick Start Application).

Here are some quick links to get you going:
  • To see examples of components in action, click a component name in the side panel under "Components".
  • For instructions on installing the Living Style Guide (this application) locally, click the Installation tab above.
  • To learn how to develop & contribute to WUF, visit the Development section.
  • For instructions on developing a NEW application based on WUF, view the Quick Start Application page.

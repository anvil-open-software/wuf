Welcome to the WUF Living Style Guide

The Web UI Framework (WUF) is a framework based on Angular designed to help developers quickly and easily create applications built on a modern platform, web standards, and best practices.

The WUF is comprised of the following 3 repos:

  1. The web-ui-framework repo. This repo holds the Living Style Guide (this application) as well as the Library of Components.
  2. The Quick Start Application (Minimal) repo.
  3. The Quick Start Application (Complete) repo.

This application, the Living Style Guide, will serve as a living, self-documenting application that will introduce developers to WUF, demonstrate the use of various components available within the Library of Components, and explain how to build applications based on WUF (i.e., the Quick Start Application).

Here are some quick links to get you going:
  • To see examples of components in action, click a component name in the side panel under "Components".
  • For instructions on installing the Living Style Guide (this application) locally, click the Installation tab above.
  • To learn how to develop & contribute to the WUF, visit the Development section.
  • For instructions on developing a NEW application based on WUF, view the Quick Start Application (Minimal) page.

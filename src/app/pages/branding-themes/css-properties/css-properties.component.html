CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are generated through the theme creation process and therefore available to use throughout your application should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

Colors

These values can be overwritten using the wuf-theme-colors() method. See KION Theme System documentation for more information.

Logo

These values can be overwritten using the wuf-theme-logo() method. See KION Theme System documentation for more information.

CSS Custom property Description Value
--wuf-header-logo-width The width of the logo image. This value should not exceed 130px. 120px
--wuf-header-logo-height The height of the logo image. This value should not exceed 40px. 25px
--wuf-logo-url The URL for the logo. Using an svg image format is highly recommended. Note the use of base-href on the KION Theme System page under the Define Brand Logo section. url('~@anviltech/wuf-web-assets/assets/images/logo.svg')

Branding & Themes

The KION Theme System uses an organized approach for applying color, font styles, and other decorative styles to the UI. This system is primarily based on the convention established by Material Design but is different enough to warrant its own documentation.

The KION Theme system includes a color system, a set of SCSS convenience functions used to quickly generate a collection of theme-related CSS custom properties, and various other overrides that can be used to customize the style and appearance of an application.

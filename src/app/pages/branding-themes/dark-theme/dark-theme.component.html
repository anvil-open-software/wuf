KION Dark Theme Mechanism

While the theme system allows a developer to generate a collection of CSS custom properties grouped together in a "theme", an additional "dark theme" mode can be applied in addition to and on top of a given theme.

The @anviltech/wuf-web-assets package includes a "dark theme" that provides some pre-set styles for page content, table styles, and the like. Rather than replacing a theme, the dark theme styling properties enhances the theme by leveraging a theme's color palette such that the content area's background uses a dark color value and the foreground colors uses a light color value. The end effect is an "inverted" color scheme.

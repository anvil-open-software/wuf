CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

The following CSS custom properties are global in nature. In other words, they are not bound to any specific component, per se. Therefore, we are documenting these properties under the "Layout" section even though some of these are not directly applicable to layouts.

Colors

These values can be overwritten using the wuf-theme-colors() method. See KION Theme System documentation for more information.

CSS Custom property Description Value
body
--wuf-body-bg The background color for a page's content area var(--wuf-color-gray-100)
border
Border properties are used to apply border values to all elements with borders. <hr> and <table> tags, for example, all use these values for decorating their borders.
--wuf-border-width The width used for borders 1
--wuf-border-color The color used for borders var(--wuf-color-gray-300)
--wuf-border-radius The default corner radius value used for SCSS border-radius mixins. .25rem
page layout
--wuf-page-width The "page" is the outmost container for all application pages, including the app's chrome, navigation elements, header, footer, etc. It's recommended to keep this outmost wrapper at 100% width of the browser's window. 100%
--wuf-page-padding The padding, if any, to show between the browser window's chrome and the "page" wrapper. 0
--wuf-page-margin The margin, if any, to show between around the "page" wrapper. Setting this value to "0 auto" will cause page content to be placed in the center of the page. 0
--wuf-content-padding The padding to show inside the main content area. 20px
footer
--wuf-footer-height The height of footers (the default value for both content footer and sidebar footer). 35px
content footer
--wuf-content-footer-height The height of the content footer. var(--wuf-footer-height)
--wuf-content-footer-bg-color The background color of the content footer. rgba(255,255,255,.9)
primary color
--wuf-header-color The text color for header content. var(--wuf-color-primary-lighter-50)
--wuf-header-bg-color The background color for the header. var(--wuf-color-primary)
--wuf-header-height The header height. 60px
--wuf-header-link-color The color to use for links in the header. var(--wuf-color-primary-lighter-50)
--wuf-header-link-hover-color The color to use for header link hover. var(--wuf-color-primary-lighter-60)
Sidebar
For additional sidebar-related CSS custom properties view documentation for Navigation and Navigation Footer.
--wuf-sidebar-width The initial width of the sidebar. Note that the user can override this value using the sidebar resizer drag handle. 230px
--wuf-sidebar-color The text color of content displayed in the sidebar. var(--wuf-color-secondary-lighter-50)
--wuf-sidebar-bg-color The background color for the sidebar. var(--wuf-color-secondary)
--wuf-sidebar-scroll-track-bg-color The sidebar scroll bar's track color (Note that Firefox does not support scrollbar styling) var(--wuf-color-secondary-darker-5)
--wuf-sidebar-scroll-thumb-bg-color The sidebar scroll bar's thumb color (Note that Firefox does not support scrollbar styling) var(--wuf-color-secondary-lighter-20)
--wuf-sidebar-resizer-width The width of the sidebar resizer bar 6px
--wuf-sidebar-resizer-bg-color The background color of the sidebar resizer bar. var(--wuf-color-gray-300)
--wuf-sidebar-resizer-draghandle-color The draghandle color for the sidebar resizer bar. var(--wuf-color-gray-100)
Tabs
Tabs are a component local only to the Living Style Guide. The following values are provided in the case a developer would like to copy the local tab component from this app for use in other applications.
--wuf-tabs-color Tab label text color. var(--wuf-body-color)
--wuf-tabs-wrap-bg-color The background color against which the tabs appear. This is not the color of the tabs themselves; rather it is the color of the parent container for the tabs. By default it matches the color of the content toolbar. var(--wuf-color-gray-200)
Pagination Links
--wuf-pagination-color Text color for pagination links. var(--wuf-link-color)
--wuf-pagination-bg-color Background color for normal pagination links. var(--wuf-color-white)
--wuf-pagination-border-color Border color for normal pagination links. var(--wuf-border-color)
--wuf-pagination-hover-color Text color for pagination links on hover. var(--wuf-link-hover-color)
--wuf-pagination-hover-bg-color Background color for pagination links on hover. var(--wuf-color-gray-200)
--wuf-pagination-hover-border-color Border color for pagination links on hover. var(--wuf-color-gray-300)
--wuf-pagination-active-color Text color for active pagination links. var(--wuf-color-primary)
--wuf-pagination-active-bg-color Background color for active pagination links. var(--wuf-color-gray-200)
--wuf-pagination-active-border-color Border color for active pagination links. var(--wuf-color-primary)
--wuf-pagination-disabled-color Text color for disabled pagination links. var(--wuf-color-gray-600)
--wuf-pagination-disabled-bg-color Background color for disabled pagination links. var(--wuf-color-white)
--wuf-pagination-disabled-border-color Border color for disabled pagination links. var(--wuf-color-gray-300)
Scrollbars
--wuf-scroll-track-bg-color The scrollbar track color for scrollable areas. Only compatible with webkit-based browsers (like Chrome and Safari). var(--wuf-color-gray-200)
--wuf-scroll-thumb-bg-color The scrollbar thumb color for scrollable areas. Only compatible with webkit-based browsers (like Chrome and Safari). var(--wuf-color-gray-400)

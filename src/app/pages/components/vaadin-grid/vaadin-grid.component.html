Vaadin Grid

Vaadin Grids are a power grid/table component based on Polymer. Vaadin Grids have a very robust feature set and are, therefore, the recommended grid component for use with WUF.

Example Vaadin Grid

[When/Why to Use]

The @kion/vaadin-elements from Common Components doesn't provide any packages itself. Instead, it installs various package dependencies required for Vaadin components, like the Vaadin Grid. It also declares a dependency on the @kion/kg-grid component, which includes KION-specifc styling for Vaadin Grids. By installing @kion/vaadin-elements instead of the Vaadin Grid components (and required dependencies) directy, you are ensuring that your Vaadin component versions have been tested for compatibility with WUF and that style dependencies will be available.

Similary, the @kion/paper-elements component from Common Components provides all of the controls used within Vaadin Grid for column sort icon, column filters, etc.

Installation

				$ yarn add @kion/kg-poly-vaadin-elements @kion/kg-poly-paper-elements

Then add the following lines to your application's app.module.ts file:

				
// Import polymer controls that will be used by Vaadin Grid
import '@polymer/polymer/lib/elements/dom-if.js';
import '@polymer/paper-button/paper-button.js';
import '@polymer/paper-input/paper-input.js';
import '@polymer/paper-checkbox/paper-checkbox.js';
import '@polymer/iron-ajax/iron-ajax.js';
import '@polymer/iron-icons/iron-icons.js';
import '@polymer/iron-icon/iron-icon.js';

// Import Vaadin Grid components
import '@vaadin/vaadin-grid/vaadin-grid.js';
import '@vaadin/vaadin-grid/vaadin-grid-selection-column.js';
import '@vaadin/vaadin-grid/vaadin-grid-column';
import '@vaadin/vaadin-grid/vaadin-grid-sorter';
import '@vaadin/vaadin-grid/vaadin-grid-filter';
import '@kion/kg-poly-grid-styles'

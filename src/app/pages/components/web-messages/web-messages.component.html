CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

CSS Custom property Description Value
--wuf-message-success-bg-color Color to use for info message background. var(--wuf-color-success)
--wuf-message-success-color Color to use for success message text. var(--wuf-color-white)
--wuf-message-success-link-color Color to use for success message links appearing in message body. var(--wuf-color-success-darker)
--wuf-message-info-bg-color Color to use for info message background. var(--wuf-color-info)
--wuf-message-info-color Color to use for info message text. var(--wuf-color-white)
--wuf-message-info-link-color Color to use for info message links appearing in message body. var(--wuf-color-info-darker)
--wuf-message-warning-bg-color Color to use for warning message background. var(--wuf-color-warning)
--wuf-message-warning-color Color to use for warning message text. var(--wuf-color-warning-darkest)
--wuf-message-warning-link-color Color to use for warning message links appearing in message body. var(--wuf-color-warning-darker)
--wuf-message-danger-bg-color Color to use for danger/alert message background. var(--wuf-color-danger)
--wuf-message-danger-color Color to use for danger/alert message text. var(--wuf-color-white)
--wuf-message-danger-link-color Color to use for danger/alert message links appearing in message body. var(--wuf-color-danger-darker)
--wuf-message-close-color Color to use for message color button. var( --wuf-color-gray-800, #424242 )
--wuf-message-close-hover-color Color to use for message color button on hover. var( --wuf-color-gray-800, #424242 )
--wuf-message-close-shadow-color Color to use for message close button shadow. var( --wuf-color-white, #ffffff )

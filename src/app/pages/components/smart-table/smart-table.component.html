DEPRECATION NOTICE!

This component has been deprecated in favor of Vaadin Grids. There is currently some debate about whether Vaadin will be the officially supported grid of choice or whether we will return to updating the Smart Table component. While Smart Table is a native Angular component, Vaadin grid (a polymer component) has a more robust feature set. Consider this a fair warning if deciding to adopt Smart Tables.

Property Type Default Description
Other Properties

filterByLabel string 'filter by' The text prefixed to the input name appearing in the filter input as placeholder text.
hover boolean false Whether or not to display a highlight color on the row when the user's mouse passes over the row.
noDataMessage string 'No data found' The text to display when no data is loaded into the table.
valign string 'middle' Sets the vertical alignment of every cell in the table. Possible values are top, middle, bottom.
width string '100%' Takes any valid CSS values for the width property. By default the table will stetch to 100% width of its parent container. By setting the value to "auto" or an empty string ("") the table width will be determined by the contents. Note that if the contents of the table or the number of columns requires the table to stretch wider than 100% of available width of the containing element, the table will be as wide as it needs to be regardless of the value of this setting.

CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

CSS Custom property Description Value

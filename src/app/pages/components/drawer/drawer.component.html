WufDrawerService Methods

Method Description
WufDrawerService.show(id: string) Shows the drawer. Takes unique ID of drawer to show.
WufDrawerService.hide(id: string) Hides the drawer. Takes unique ID of drawer to hide.

<wuf-drawer> Properties

Property Type Default Description
header string none The title to display in the header section of the drawer.
id string none The unique identifier used to identify the drawer for show/hide methods.

CSS Custom Properties

CSS Custom property Description Value
--wuf-drawer-width The width of the drawer when visible. Takes any CSS value for width. 300px
--wuf-drawer-header-bg-color The background color for the titlebar of the drawer. It's recommended to use an rgba value so that it is compatible with Dark Theme mode. rgba(0,0,0, .05)
--wuf-drawer-body-bg-color The background color for the body of the drawer. var(--wuf-body-bg, var(--wuf-color-gray-100, #404040) )
--wuf-drawer-shield-bg-color The background color for the overlay "shield" appearing behind the drawer. rgba(0,0,0, .2)

Navigation

The following example loads <wuf-navigation> inside of a container with a fixed width and color background in order to simulate the appearance of the component inside a sidebar.

By default <wuf-navigation> assumes it is positioned on the left side of the layout and orients itself vertically. By changing the value of the "position" attribute, however, we can instruct the <wuf-navigation> to orient itself horizontally for positioning at the top of the layout.

By default navigation items place their icons to the left of the text label. To place the icons on top, change the "iconPosition" attribute to "top".

Orientation

By default <wuf-navigation> assumes it is positioned on the left side of the layout and orients itself vertically. By changing the value of the "position" attribute, however, we can instruct the <wuf-navigation> to orient itself horizontally for positioning at the top of the layout. It is strongly recommended to use the configuation service (via WufConfigurationService) to supply <wuf-navigation> with the value for the position attribute.

The position attribute from <wuf-navigation> will propagate down to the children <wuf-navigation-link> elements.

Icon Position

By default icons are positioned on the left side of a navigation item. To change this behavior, pass a value to the "iconPosition" attribute. Possible values include "top", "left".

The iconPosition attribute from <wuf-navigation> will propagate down to the children <wuf-navigation-link> elements.

Properties

Property Type Default Description
dataUrl string None The url of an API to use to retrieve navigation data. The service must return an object in the navigation object format.
data object None A navigation object with navigation data.
activeOptions string '{{'{'}}exact: true{{'}'}}' The position of the navigation element, either {{'{'}}exact: true{{'}'}} or {{'{'}}exact: false{{'}'}}. This will put the link active if needed when the path is exact or almost that. This property is related to routerLinkActiveOptions of a link.
position string 'left' The position of the navigation element, either 'top' or 'left'. A value of 'top' orients the navigation component horizontally. A value of 'left' orients the navigation component vertically.
iconPosition string 'left' The position of navigation items' icon relative to the text.
alignment string 'left' The alignment of navigation items within the header. This only has an effect when position="top". Possible values are "left" (default), "center", and "right".

CSS Custom Properties

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

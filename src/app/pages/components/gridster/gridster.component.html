Attributes

The following attributes can be applied to the <wuf-gridster> component:

Property Type Default Description
items array none An array of gridster items to show.
outerMargin boolean true Whether or not to display an outer margin around the grid. It generally looks better to include a margin when dragging items. However, there may be some circumstances under which an outer margin may cause too much padding in a certain context and it is not easy to otherwise remove the outermost padding/margin.
draggable boolean true Whether or not grid items are draggable by default. This value can be overridden at the item level. When set to false, neighboring items will not be pushed out of the way on resize.
resizable boolean true Whether or not grid items are resizable by default. This value can be overridden at the item level.

CSS Custom Properties

CSS Custom property Description Value
--wuf-card-bg-color The background color used for card elements, including gridster items. var(--wuf-color-white, #ffffff)

WufGridsterService

A WufGridsterService service is available for adding/deleting gridster items. An RxJs subject is also provided, allowing your application to subscribe to update events.

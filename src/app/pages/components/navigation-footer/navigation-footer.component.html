You can find the <wuf-sidebar-footer> in src/app/internal/layouts/main/main.component.html.

By default the navigation footer displays copyright information.

You can override the default behavior of <wuf-sidebar-footer> in one of three ways, using:

  1. Content projection
  2. Passing a data array in the data attribute
  3. Passing a dataUrl in the dataUrl attribute

CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

CSS Custom property Description Value
--wuf-sidebar-footer-height The sidebar footer's height. var(--wuf-footer-height)
--wuf-sidebar-footer-bg-color The sidebar footer's background color. var(--wuf-color-primary-darker-15)
--wuf-sidebar-footer-color The sidebar footer's text color. var(--wuf-color-primary-lighter-10)
--wuf-sidebar-footer-hover-color The sidebar footer's link hover color. var(--wuf-color-primary-lighter-20)
--wuf-sidebar-footer-hover-bg-color The sidebar footer's hover background color. var(--wuf-color-primary-darker-25)
--wuf-sidebar-footer-active-color The background color to display on active sidebar footer items. var(--wuf-color-white)
--wuf-sidebar-footer-active-bg-color The text color to display on active sidebar footer items. var(--wuf-color-secondary)

For additional sidebar-related CSS custom properties view documentation for Layout and Navigation.

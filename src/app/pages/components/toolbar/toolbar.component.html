Installation

The toolbar component is included inside @anviltech/wuf-ang-layout, which is already included in every instance of a Quick Start-based application.

Usage

The <wuf-toolbar> tag is created for you and automatically added to the top of an application page when you use the <wuf-content-header> component.

The <wuf-toolbar> can be used elsewhere, including within page content, by applying the following to your page:

The toolbar supports two different content slots:

  • toolbarContent. Content placed in this slot appears on the left side of the toolbar, left-aligned. This slot is used for the breadcrumb trail, for example, when used at the page level.
  • utilities. Content placed in this slot appears on the right side of the toolbar, right-aligned. This slot is used for utility links, for example, when used at the page level.

CSS Variables (i.e., CSS Custom Properties) Reference

The following theme-related CSS custom properties are available to use throughout your application's CSS files should you decide to use them to further customize your brand's appearance. By default, the values of these properties are set to use the Dematic brand theme. However, you can easily override all of the following default value with your own theme values using the SCSS methods definded in the KION Theme System documentation.

CSS Custom property Description Value
--wuf-toolbar-color Toolbar text color. var(--wuf-body-color)
--wuf-toolbar-bg-color Toolbar background color. var(--wuf-color-gray-200)
--wuf-toolbar-height Toolbar height. 40px
--wuf-toolbar-link-color Toolbar link text color. var(--wuf-color-gray-600)
--wuf-toolbar-link-hover-color Toolbar link text hover color. var(--wuf-color-gray-700)
--wuf-toolbar-button-color Toolbar button text color. var(--wuf-color-gray-600)
--wuf-toolbar-button-bg-color Toolbar button background color. var(--wuf-color-gray-200)
--wuf-toolbar-button-bg-hover-color Toolbar button hover background color. var(--wuf-color-gray-100)
